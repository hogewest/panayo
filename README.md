# Panayo

イ゛エ゛ニ゛カ゛エ゛ッチ゛ャッタ゛ノ゛？

## Installation

Add this line to your application's Gemfile:

    gem 'panayo'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install panayo

## Usage

    require 'panayo'

    Panayo.say('いえにかえっちゃったの？') # => "イ゛エ゛ニ゛カ゛エ゛ッ゛チ゛ャ゛ッ゛タ゛ノ゛？"
